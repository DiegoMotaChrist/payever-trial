## Description

Projeto para avaliação.

## Installation

Certifique-se de ter uma instâncias locais dos seguintes serviços abaixo e suas respectivas credenciais ou altere no arquivo de ambiente (.env):

- MongoDB

- RabbitMQ

```bash
$ npm install
```

## Running the app

```bash

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# tests
$ npm run test

# test coverage
$ npm run test:cov
```
